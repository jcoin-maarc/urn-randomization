CLI¶

User management¶

Users for the Urn Randomization app can only be added via its command line interface.

urand_gui/cli.py create_db¶

Initializes flask app DB with available studies & creates user tables

urand_gui/cli.py create_db [OPTIONS]

urand_gui/cli.py add_user¶

Adds a user

urand_gui/cli.py add_user [OPTIONS] UNAME EMAIL

Arguments

UNAME¶

Required argument

EMAIL¶

Required argument

urand_gui/cli.py list_users¶

Lists available users

urand_gui/cli.py list_users [OPTIONS]

urand_gui/cli.py delete_user¶

Deletes user linked to email

urand_gui/cli.py delete_user [OPTIONS] EMAIL

Arguments

EMAIL¶

Required argument

Urn randomization¶

Urn Randomization app may be accessed via its command line, permitting integration with batch/ scheduler jobs.

urand/cli.py¶

Perform urn randomization as described by Wei (1978)

urand/cli.py [OPTIONS] COMMAND [ARGS]...

Options

-s, --study-name <study_name>¶

Required Name of study

dummy-study¶

Populate a study with dummy data

urand/cli.py dummy-study [OPTIONS]

Options

--n_participants <n_participants>¶

Required Study size

--seed <seed>¶

Required Seed

export¶

Export study history to OUTFILE

urand/cli.py export [OPTIONS] OUTFILE

Arguments

OUTFILE¶

Required argument

randomize¶

Randomize new participant

urand/cli.py randomize [OPTIONS]

Options

--id <id>¶

Participant ID

-u, --user <user>¶

Username

upload-participants¶

Export study history to OUTFILE

urand/cli.py upload-participants [OPTIONS] INFILE

Arguments

INFILE¶

Required argument
